El componente `f24-copy` permite a los usuarios copiar texto o contenido al portapapeles de forma rápida con un solo clic, utilizando componentes de Angular Material y el CDK.

### Importación

Para usar este componente, impórtalo en tu componente o módulo:

```typescript
import { F24Copy } from '@f24/layout';
```

### Creación

Para crear este elemento usa esta etiqueta

```html
<f24-copy [text]="'Show text'" [copy]="'Copy text'"/>
```
